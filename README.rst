Snippets
========

``inc/roman-numerals.ly``
   | Convert chord names to roman numerals
   | 将和弦名称转换为罗马数字（和弦级数）

``inc/etude.ly``
   | Generate melodies and harmonies in 12 keys for practice
   | 为曲子生成 12 个调的旋律、和声供练习使用
